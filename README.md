# Kishore Kumar — React Portfolio

## Quick Start
```bash
npm install
npm run dev   # http://localhost:5173
```

## Structure
```
src/
├── components/index.jsx   ← All UI components
├── data/portfolioData.js  ← All resume content (edit this)
├── hooks/usePortfolioHooks.js ← cursor, typewriter, Three.js, scroll-reveal
├── styles/globals.css     ← All CSS
└── App.jsx                ← Root component
```

## Customise
Edit `src/data/portfolioData.js` to update your resume content.
